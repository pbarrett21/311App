Admin Guide

  • System Requirements
    • Windows, Mac, Linux, etc. machine to run on
    • Memory, RAM, CPU needs are negligible
    • Node Package Manager (NPM)
    • MongoDB server
    • Node.js
  • Installation
    1. Download source code to machine by pulling or cloning from the 311App Repository
    2. Download MongoDB if it is not already installed
    3. Download NPM if it is not already installed
    4. Navigate to 311Application/Angular311 in terminal and run npm install
    5. Start the Mongo database by running mongo or whichever command your version of MongoDB requires in the terminal
    6. Navigate to 311Application/server in terminal and run node app.js command
    7. Navigate back to the 311Application/Angular311 directory and run ng serve
    8. Using a web browser, go to http://localhost:4200 to bring up the 311App and verify the login/register page displays
  • Uninstallation
    • Remove source files from machine
    • (Optional) Remove Node, NPM, and MongoDB
