User Guide

  • Create Account
    1. From homepage, navigate to Sign Up
    2. Input full name
    3. Input valid email
    4. Input password
    5. Input zipcode
    6. Select Sign Up
  • Login to Account
    1. From homepage, navigate to Sign In
    2. Enter email previously registered during Sign Up
    3. Enter password
    4. Click Sign In button or press enter
  • View Posts
    1. From homepage, log in
    2. If email and password are correct, posts from user's zip code will appear
  • Create New Post
    1. Login from homepage if not already logged in
    2. Scroll to the bottom of the page and click Post New Issue button
    3. Fill in prompts for Post Title, Post Test, and Zip code and click Post button
